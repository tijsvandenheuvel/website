How to make a website

Step 1. make a html page

1.1. Create a new file
1.2. Call it index.html
1.4. Open it in Visual Studio code
1.5. Type "!" + "tab" this creates the html scaffolding
1.6. Write something in between the body tags
1.7. Open the file in the browser
1.8. Tada

Step 2. publish your website

2.1. Go to github.com
2.2. Create an account
2.3. Create a new public repository
2.4. Follows these instructions to setup git locally (use https)

2.4.1. open a terminal in your folder
2.4.2. run these commands:
2.4.3. $ git init
2.4.4. $ git add .
2.4.5. $ git commit -m "first commit"
2.4.6. $ git branch -M main
2.4.7. $ git remote add origin https://github.com/your-user-name/your-repository-name.git
2.4.8. $ git push -u origin main
2.5. Go to your repository settings -> Pages
2.6. Select your main branch and click Save
2.7. Go to https://your-user-name.github.io/your-repository-name
2.8. Tada

Step 3. update your website

3.1. make a change to your index.html file
3.2. save changes, if your IDE doesn't do this automatically
3.3. open a terminal in your folder
3.4. run these commands:

3.4.1. $ git commit -m "second commit"
3.4.2. $ git push -u origin main
